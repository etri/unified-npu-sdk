# QB SDK Branch

이 브랜치는 **QB NPU 전용 지원 SDK**를 개발하고 정리하기 위한 작업 공간입니다.

주요 목적은 다음과 같습니다.

- QB 백엔드 전용 빌드/런타임 지원
- QB SDK 연동에 필요한 예제와 설정 관리
- Unified SDK 구조 안에서 QB 전용 기능 분리 및 확장

This branch is intended for development of the **QB NPU specific SDK support**.

Its main goals are:

- Backend-specific build and runtime support for QB
- Examples and configuration required for QB SDK integration
- Isolated development of QB-specific features within the Unified SDK structure
