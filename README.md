# RBLN SDK Branch

이 브랜치는 **RBLN NPU 전용 지원 SDK**를 개발하고 정리하기 위한 작업 공간입니다.

주요 목적은 다음과 같습니다.

- RBLN 백엔드 전용 빌드/런타임 지원
- RBLN SDK 연동에 필요한 예제와 설정 관리
- Unified SDK 구조 안에서 RBLN 전용 기능 분리 및 확장

This branch is intended for development of the **RBLN NPU specific SDK support**.

Its main goals are:

- Backend-specific build and runtime support for RBLN
- Examples and configuration required for RBLN SDK integration
- Isolated development of RBLN-specific features within the Unified SDK structure
